## Step 5：数据到图形的映射（Represent）

### 设计原则

数据可视化的核心是"用视觉通道（位置、颜色、大小、形状）映射数据特征"。本项目针对5个竞赛任务，依据数据类型和分析目标，选择以下图表类型：

---

### 图表选型说明

#### Task 1：职位差异化分析

| 分析内容 | 选用图表 | 选择理由 |
|----------|----------|----------|
| 学历 vs 平均薪资 | **水平条形图** | 学历为有序类别，条形图可直观比较各类别均值大小 |
| 经验 vs 平均薪资 | **水平条形图** | 同上，经验年限有自然排序，从低到高排列符合认知习惯 |
| 公司类型 vs 薪资 | **垂直柱状图** | 类别数量适中，垂直展示便于标注数值 |
| 学历×经验 联合分析 | **热力图（Heatmap）** | 两个分类变量的交叉关系，颜色深浅直观展示薪资高低 |

**视觉通道映射：**
- 位置（x/y轴）→ 薪资数值
- 颜色深浅 → 热力图中薪资高低
- 长度 → 条形图中薪资均值

#### Task 2：职位画像

| 分析内容 | 选用图表 | 选择理由 |
|----------|----------|----------|
| 薪资整体分布 | **直方图 + KDE曲线** | 展示连续变量的频率分布，KDE曲线平滑化分布趋势 |
| 各城市薪资对比 | **箱线图（Box Plot）** | 同时展示中位数、四分位数、极值，信息密度高 |
| 行业×城市招聘量 | **热力图** | 二维类别交叉，颜色展示招聘密度 |
| 学历维度画像 | **雷达图（Radar Chart）** | 多维度对比同一实体的各项指标，适合"画像"场景 |

**视觉通道映射：**
- 位置 → 薪资数值 / 频率
- 颜色 → 行业/城市分类
- 形状（箱体） → 薪资分布区间

**视觉通道映射：**
- 位置 → 特征重要性分数 / 薪资数值

| 分析内容 | 选用图表 | 选择理由 |

- 位置 → 薪资排名

## Step 6：美化和装饰（Refine）

### 整体设计原则

> **"图表不只是数据的容器，更是信息的传达媒介。"**

本项目遵循以下美化原则：

1. **一致性**：全项目统一配色方案、字体、图表风格
2. **简洁性**：去除不必要的网格线、边框、装饰元素
3. **可读性**：关键数值直接标注在图表上，减少读者的认知负担
4. **中文适配**：使用 SimHei / Microsoft YaHei 字体，避免中文乱码

---

### 配色方案

| 用途 | 颜色 | 色值 |
|------|------|------|
| 主色（学历/经验） | 钢蓝色 | `steelblue` |
| 辅色（对比） | 珊瑚色 | `coral` |
| 强调（重点数据） | 海绿色 | `seagreen` |
| 热力图 | 蓝→红渐变 | `coolwarm` |
| 聚类着色 | Set2 调色板 | 柔和多色 |

**选色理由：** 蓝色系在数据可视化中被广泛接受，视觉上偏"中性/专业"；热力图使用冷暖色渐变，直觉上冷色=低值、暖色=高值。

---

### 字体与文字

```python
plt.rcParams['font.sans-serif'] = ['SimHei', 'Microsoft YaHei']
plt.rcParams['axes.unicode_minus'] = False  # 解决负号显示问题
```

- 标题：加粗，字号14-16
- 坐标轴标签：字号12
- 数值标注：字号9-10，直接显示在柱/条末端

---

### 布局与尺寸

| 图表类型 | 尺寸设置 | 理由 |
|----------|----------|------|
| 单图 | `figsize=(10, 6)` | 标准宽高比，适合文档插入 |
| 双图并排 | `figsize=(14, 5)` | 等高展示对比 |
| Dashboard | `figsize=(18, 12)` | 多图联合展示需要更大画布 |

---

### 数值标注规范

```python
# 条形图数值标注示例
for bar, val in zip(bars, values):
    ax.text(val + 0.2,                    # 标注位置：略偏右
            bar.get_y() + bar.get_height()/2,  # 垂直居中
            f'{val:.1f}K',                # 保留1位小数+单位
            va='center', fontsize=9)
```

---

### 输出规范

- 格式：PNG（通用，适合文档插入）
- 分辨率：`dpi=150`（清晰但文件不过大）
- 边距处理：`bbox_inches='tight'`（自动裁剪多余白边）
- 保存路径：统一保存至 `chinavis2024/figures/`

---

### 美化前后对比

| 方面 | 美化前 | 美化后 |
|------|--------|--------|
| 字体 | 默认英文字体，中文乱码 | SimHei，中文正常显示 |
| 数值 | 需对照坐标轴估读 | 直接标注在图上 |
| 颜色 | 默认蓝色单色 | 按语义分配配色 |

## Step 7：交互（Interact）

### 交互实现方案

本项目静态分析部分使用 **Matplotlib**（Python库）完成，图表为PNG静态图像。

在此基础上，额外使用 **ECharts**（百度开源JavaScript图表库）实现了交互版本，主要交互功能包括：

---

### 实现的交互功能

| 功能 | 实现方式 | 说明 |
|------|----------|------|
| **Tooltip 悬浮提示** | ECharts 内置 | 鼠标悬停显示详细数值 |
| **图例筛选** | ECharts 内置 | 点击图例显示/隐藏对应系列 |
| **数据缩放** | dataZoom 组件 | 拖动滑块查看局部数据 |
| **切换维度** | 下拉选择器 | 切换查看不同字段的薪资分布 |

### 交互文件

交互可视化文件：`chinavis2024/interactive_dashboard.html`

使用方法：直接用浏览器打开该文件即可，无需服务器，无需安装任何依赖。

---

### 技术选型理由

| 技术 | 优点 | 本项目使用 |
|------|------|-----------|
| Matplotlib | Python原生，适合批量生成 | ✅ 静态图表（20张PNG） |
| ECharts | 交互丰富，纯前端无依赖 | ✅ 交互大屏（HTML） |
| D3.js | 高度定制化 | ❌ 学习成本高，未使用 |
| Plotly | Python+交互结合 | ❌ 依赖较重，未使用 |

---

### 交互设计说明

交互版本实现了以下可视化面板：

1. **薪资分布直方图**：可拖动缩放查看局部区间
2. **学历薪资对比**：鼠标悬停显示均值/中位数/样本量
3. **经验薪资趋势**：折线图展示经验增长与薪资增长关系
4. **城市招聘量气泡图**：气泡大小=招聘量，悬停显示城市编码和详情
5. **维度切换器**：一键切换查看不同维度的薪资分布
